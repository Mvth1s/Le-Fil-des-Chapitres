# Le Fil des Chapitres

Application web pour suivre et gérer ses webtoons lus. Ajoutez, modifiez et organisez vos webtoons avec leurs métadonnées (nom, auteur, genres, chapitres lus/totaux, lien de lecture, statut, etc.). Utilise une base de données et Docker pour un déploiement simplifié.

**Technologies** : Nuxt 3 (Vue 3), AdonisJS, PostgreSQL, Docker.
